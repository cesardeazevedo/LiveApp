# LiveApp
Experiment with [LiveScript](http://livescript.net/) + Node.JS + Express + Gulp

# Get Started

Install the npm modules:

`npm install`

and then compile gulpfile.ls

`lsc -c gulpfile.ls`

# Run

Running with gulp:

`gulp`

Or run with npm:

`npm start`
